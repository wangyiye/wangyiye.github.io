REACT脚手架

创建的项目下的文件夹详情

  • public
    _ index.html(SPA各组件最后渲染完后的结果都放入此页面的#root盒子中) ![Alt

    _ xxx.html(MPA/SPA 存放编译页面的模板)
    _公共资源（直接基于link的方式导入到模板当中不是基于webpack合并到一起，项目中某些模块不支持CommonJS规范/es6module不能import导入，只能在此引入使用）

  • src （整个项目的源码存放目录）

    • index.js （当前项目入口，webpack从这个文件开始导入打包（MPA中创建多个入口文件即可）
    • api
    • store REDUX公共状态管理
    • assets 存储公共i资源
    • route 路由管理
    • utils （公共JS模块）
    • components 公共组件

  • package.json （依赖清单）

    • "react": "^16.10.2", react核心，提供 状态/属性/组件/生命周期等 "react-dom": "^16.10.2", JSX语法渲染成为真实DOM显示在浏览器中 "react-scripts": "3.2.0" webpack配置项，隐藏在node_module中，脚本执行通知webpack打包编译 
    • > 执行 yarn eject 前提交历史版本不可逆操作
      • eject
        • babel-preset-react-app 解析JS语法
    • 修改默认域名端口

  • config

    • webpack.config.js 修改合并压缩依赖的插件和加载器配置
      • less重新配置 下载 less less-loader

JSX

javascript 和 xml(html) 虚拟的DOM

  • 只能有一个根元素节点 不要用HTML和BODY
  • 基于{}或者JS表达式
    • 可以识别null/undefined
    • 不能直接使用引用数据类型,除了数组，数组会转化为字符串
    • 可以使用表达式返回有效的数据值或者返回JSX元素
  • JSX元素设置样式
    • 设置样式类 className
      • className =’box’ 值是字符串
      • style={color} 值是对象
